# Guía de contribución

Gracias por contribuir a esta referencia educativa del mercado financiero boliviano.

## Datos y fuentes

Toda propuesta que cambie o agregue un instrumento debe indicar una fuente verificable y la fecha de consulta. Conserva las condiciones importantes: moneda, plazo, liquidez, tasa variable o escalonada, comisiones y riesgos relevantes.

No presentes rendimientos como garantizados ni escribas contenido que pueda interpretarse como asesoramiento financiero, legal o tributario.

## Pull requests

Mantén cada cambio acotado, explica el motivo y actualiza el README cuando cambie el alcance público del proyecto. No publiques datos personales, cuentas, documentos privados ni información no autorizada de una entidad.
